# Notebooks

Este directorio contiene notebooks Jupyter para análisis exploratorio y experimentación.

## Estructura

- `exploracion/`: Análisis exploratorio de datos
- `experimentos/`: Experimentación con modelos ML
- `analisis_app.ipynb`: Notebook de referencia para la app Streamlit

## Uso

```bash
jupyter lab notebooks/
```

## Notas

- No guardar notebooks con datos sensibles
- Usar rutas relativas desde la raíz del proyecto
